# 快速开始 - 统一安全扫描平台

## 🎉 系统已成功运行！

现在可以开始使用系统了。以下是快速操作指南：

## 📋 第一步：创建项目

1. 点击左侧菜单 **"项目管理"**
2. 点击右上角 **"+ 新建项目"** 按钮
3. 填写信息：
   - 项目名称：例如 "我的Web应用"
   - 项目类型：选择类型
   - 描述：（可选）
4. 点击 **"确定"** 保存

## 📋 第二步：创建扫描任务

1. 点击左侧菜单 **"扫描任务"**
2. 点击右上角 **"+ 新建扫描"** 按钮
3. 选择：
   - **项目**：选择刚才创建的项目
   - **扫描类型**：
     - **SAST** - 静态代码扫描（推荐开始）
     - **SCA** - 依赖漏洞扫描
     - **容器扫描** - Docker镜像扫描
     - **RASP** - 运行时安全扫描
   - **目标**：根据类型填写路径或名称
4. 点击 **"开始扫描"**

## 📋 第三步：查看结果

- 扫描完成后，在扫描任务列表中查看结果
- Dashboard会显示统计数据
- 可以查看详细漏洞信息

## ⚠️ 如果看到错误

### "获取XX失败"错误

这通常是因为：
1. **还没有创建数据**（正常，先创建项目）
2. **API连接问题**（检查后端是否运行）

### 检查后端

```bash
# 在Ubuntu上执行
# 1. 检查后端状态
docker-compose ps backend

# 2. 测试API
curl http://localhost:5000/api/health

# 3. 查看日志
docker-compose logs backend
```

## 🎯 推荐操作顺序

1. ✅ **创建项目** → 项目管理页面
2. ✅ **创建扫描任务** → 扫描任务页面（建议先试SAST）
3. ✅ **查看结果** → 扫描任务详情
4. ✅ **查看统计** → Dashboard页面
